## 📖 项目简介

> "打开冰箱不知道做什么？食材买了又浪费？新手下厨无从下手？"

**食用手册** 是一款面向厨房新手的 AI 烹饪 Agent 应用。
不同于传统菜谱 App 的被动查询，食用手册通过 **多轮对话 + 分步引导**，
像一个耐心的朋友一样，从"有什么食材"到"做好上桌"全程陪伴。

---

## ✨ 核心功能

| 功能 | 描述 |
|------|------|
| 🤖 **多轮对话推荐** | 根据你的食材、厨具、口味偏好，智能推荐最适合的菜品 |
| 📝 **分步烹饪引导** | 选定菜品后一步步教学，带进度条，不会手忙脚乱 |
| 📷 **拍照识别食材** | 拍一张冰箱照片，AI自动识别食材并推荐菜品 |
| 🎙️ **语音交互** | 做饭时手上有油？直接说话，解放双手 |
| 🧠 **烹饪记忆系统** | 记住你做过的菜、口味偏好，越用越懂你 |
| 📊 **成长体系** | 从新手到大厨，记录每一次烹饪成长 |
